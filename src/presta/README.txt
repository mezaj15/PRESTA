PRESTA

Descripción:
El Sistema de Préstamos (PRESTA) es una aplicación Java que permite a 
los usuarios solicitar préstamos y hacer un seguimiento de sus solicitudes. 
Este proyecto tiene un frontend ejecutable que permite a los usuarios 
interactuar con la aplicación de manera intuitiva.

Requisitos Previos:
Antes de ejecutar PRESTA, asegúrate de tener instalado lo siguiente

Java 8 o superior

Instalación:
Puedes descargar el archivo ejecutable JAR del github.

Uso
Una vez que PRESTA esté en funcionamiento, podrás:

Iniciar sesión como usuario registrado.
Solicitar un préstamo.
Ver el estado de tus solicitudes como admin (al mismo tiempo).
